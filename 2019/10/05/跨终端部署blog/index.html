基本思路：
将blog　整个文件git 到 github 的 私库中，这样就是对blog的 完整 备份（具体细节见下方）；
hexo 生成的静态页面存储git 在另一公库中。

关于npm全局和本地安装的区别

没有 -g npm 命令是本地安装，即在当前路径下将包安装在node_modules 文件里，而非全局路径下

以下有两个本地安装命令就是将包安装在了当前hexo创建的blog文件的 node_modules 文件

查看在当前所在路径下安装过的包

在blog 文件夹的路径下 终端命令键入npm list --depth 0 可查看本地安装的包，即node_modules 文件夹中包。
